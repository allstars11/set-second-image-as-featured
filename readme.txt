Automatically sets the featured image to the second image block's image when editing and saving a post using the Gutenberg editor.

== Description ==

Gutenberg Featured Image Setter is a simple plugin that enhances your workflow when editing posts using the Gutenberg editor. When you save a post, the plugin checks if a featured image is already defined for the post. If not, it automatically sets the featured image to be the image contained in the second image block of the post.

Please note that this plugin is designed to work with WordPress 6.0 and later versions and is intended for use on pages composed with blocks using the Gutenberg editor.

== Installation ==

1. Upload the `gutenberg-featured-image-setter` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. There are no additional settings. The plugin works automatically when editing and saving posts using the Gutenberg editor.

== Frequently Asked Questions ==

= Is this plugin compatible with older versions of WordPress? =

No, this plugin is designed to work with WordPress 6.0 and later versions that support the Gutenberg editor.
